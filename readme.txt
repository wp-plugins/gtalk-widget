== Description ==
A Gadget that brings Gtalk on your website. Members can logon and chat with their friends on gtakl while reading your blogs

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `gtalk.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the title, height and width of the widget from Wordpress Widget backend
4. Activate the widget through the 'Widget' menu in WordPress
